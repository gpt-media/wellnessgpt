# TheWellnessGPT

Evidence-led, global wellness answers, built to be read and cited by AI answer engines and to rank in local search. Independent publication.

## Architecture — global root + country editions

Built to the ratified GPT-sites standard (Wone repo: `knowledge/operations/gpt-sites-architecture.md`):

- Root domain is the default English edition (`hreflang=x-default`), USD shown.
- Country editions in subfolders: `/my/` `/sg/` `/tw/`, with local currency + brands.
- Languages in nested subfolders: `/my/ms/` `/my/zh/` `/sg/ms/` `/sg/zh/`; Taiwan `/tw/` is Traditional Chinese.
- 8 editions per article. `hreflang` (bidirectional + x-default) and self-canonical on every page. Header region/language selector; no auto IP redirects (Google i18n best practice).

The edition model lives in `src/lib/editions.ts`; brand identity + multilingual chrome in `src/lib/site.ts`. Add a country by adding a row to `EDITIONS`.

Stack: Astro static (near-zero JS = fast, clean, citeable HTML).

## Content layout

One `articles` collection, one sub-folder per edition:

    src/content/articles/<editionKey>/<slug>.md
    keys: en  my-en  my-ms  my-zh  sg-en  sg-ms  sg-zh  tw-zh

Slug is stable across editions (the hreflang key). Each file holds the localized/translated copy. Frontmatter: `title, description, publishDate, [updatedDate], [author], category, [featured], faqs[]`. The `faqs` array auto-emits FAQPage structured data.

Content is generated by the company-wide `gpt-content-cluster` skill (write → localize → translate, with a looped 4-lens quality gate).

## Local development

    npm install
    npm run dev      # http://localhost:4321
    npm run build    # static site -> dist/

Note: Astro 5 caches the content collection in `node_modules/.astro/data-store.json`. After adding/removing article files, clear caches before a verification build:

    rm -rf .astro dist node_modules/.astro && npm run build

## Editorial voice

See the voice guide in the Wone repo (`thewellnessgpt/voice-guide.md`). Evidence-led, mechanism-first, honest-to-a-fault, no hype. Names real brands fairly. "We" allowed. No em-dashes in prose. Educational, not medical advice. Independent publication (no public ownership disclosure).
